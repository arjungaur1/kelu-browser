## 📖 About The Project

The current technology ecosystem operates on a flawed premise: building visual interfaces first, and patching them with screen readers later. **Kelu (Knowledge Enabled Living Universe)** exists to completely eliminate this dependency on touch. 

Kelu is a natively voice-first web browser engineered exclusively for the visually impaired. It bypasses the visual UI layer completely, shifting the paradigm from tedious "touch-assisted reading" to **pure voice computing**. 

## ✨ Key Technical Innovations

*   **Zero-Touch UI:** Eliminates the need to tap screen coordinates. Users navigate, search, and interact with the digital world entirely through natural voice commands.
*   **100% Offline Voice-Engine:** Core voice processing occurs entirely on-device. This guarantees zero-latency, lightning-fast responses, and absolute user privacy without relying on cloud interpretation.
*   **Custom AI API Hub:** Advanced users can seamlessly plug in their preferred AI APIs (e.g., Claude, Gemini, ChatGPT) for deeply customized, human-like assistance and advanced web synthesis.
*   **Smartphone-Engineered Core:** Built under extreme constraints. The entire architecture was conceptualized, coded, and compiled natively on an Android smartphone using mobile compilers (AIDE), bypassing the traditional requirement of high-end PCs.

## 🚀 The Grand Vision

The development of the Kelu software engine is merely the foundational phase:
1.  **Phase 1 (Digital Freedom):** Establishing the Kelu browser as a seamless digital hub for the visually impaired.
2.  **Phase 2 (Physical Freedom):** Transmuting this offline voice architecture into AI Smart Glasses to provide spatial awareness and physical navigation safety in the real world.
3.  **Phase 3 (Custom OS & Hardware):** Building a custom operating system and dedicated hardware from scratch to prevent modern smartphone architectures from bottlenecking accessibility.

## 🛡️ Privacy Policy

*For Kelu, Privacy is not a feature; it is our foundation.*

As an application serving the visually impaired through voice computing, we hold user data to the strictest standards:
*   **Absolute Offline Processing:** The core voice recognition and navigation engine runs entirely offline on your local device. Your daily voice commands and audio data are **never** recorded, stored, or sent to any cloud server.
*   **API Mode Data Routing:** If a user actively chooses to use the "Custom AI API Hub," their specific queries are routed directly from their device to their chosen API provider (e.g., OpenAI, Anthropic). Kelu acts only as a bridge and does not intercept, store, or monitor these requests.
*   **Zero Tracking:** We do not use third-party analytics trackers, nor do we collect or sell user behavioral data. Your digital footprint remains yours.
